## 🚀 Deploy in 3 Steps

### Step 1: Install Dependencies
```bash
cd /home/muneebcytomate/wintogeno
npm install
```

### Step 2: Test Locally (Optional)
```bash
npm run dev
# Open http://localhost:3000
```

### Step 3: Deploy to Vercel

**Option A - Via Git (Recommended)**
```bash
git add .
git commit -m "Add DevOps portfolio with resume"
git push origin main
```
Then go to https://vercel.com/new and select your repo. Vercel auto-deploys!

**Option B - Via Vercel CLI**
```bash
npm install -g vercel
vercel --prod
```

**Option C - Run Deploy Script**
```bash
chmod +x deploy.sh
./deploy.sh
```

## 🌐 After Deployment

Once live on Vercel, you can:

1. **Set Custom Domain**
   - Go to Vercel Dashboard → Project Settings → Domains
   - Add your domain (muneeb.dev, portfolio.com, etc.)

2. **Enable Analytics** (Optional)
   - Vercel provides built-in Web Analytics
   - Enable in Project Settings → Analytics

3. **Monitor Performance**
   - View Lighthouse scores
   - Check Core Web Vitals
   - Monitor deployment history
